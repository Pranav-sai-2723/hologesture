
**Gesture-Controlled 3D Holographic System**




**Turn any 3D model into a floating interactive hologram — controlled by bare-hand gestures!**  
Built for **AMD Slingshot Hackathon 2026** under **Open Inovation** theme.

---

 ### The Problem
Students in schools & colleges still study complex 3D concepts (human skull, beating heart, working motor, DNA, engines, disassembled guns, civil structures, etc.) using flat 2D textbook diagrams.  
Result → Poor imagination, low retention, and zero real understanding.

---

###  The Solution
**HoloGest** lets you **rotate, zoom, drag, and explore** any 3D object in real-time using only your hands.  
The 3D model is converted into a **floating hologram** using the low-cost Pepper’s Ghost method 
Entire class can see and learn together — no headsets, no controllers, no touch!
This can be also used in my companies for training purposes for the employes

### Features
- Real-time hand gesture control (21 landmarks per hand)
- Smooth rotate, drag (pinch), and zoom (two-hand distance)
- Works with any 3D model (.obj / .gltf)
- Converts to 360° Pepper’s Ghost hologram

###  Technology Stack
- **Language**: Python 3
- **AI / Gesture Recognition**: MediaPipe Hands + TensorFlow Lite (XNNPACK CPU delegate)
- **3D Rendering**: Custom PyOpenGL / VTK-based viewer
- **Computer Vision**: OpenCV
- **Hologram Technique**: Pepper’s Ghost (4 synchronized views)
- **Future**: AMD Ryzen AI NPU optimization
